# Chromino
End-of-bootcamp Scala project. Browser-base, multi-player [Chromino](https://en.wikipedia.org/wiki/Chromino) server/client.

# Knowns
## Stack
- WebSocket
- Cats Effect 2
- http4s
- In-memory storage
